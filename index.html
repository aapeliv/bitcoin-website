Bitcoin block times

Open dataset containing block propagation times from the Bitcoin network from late 2018 until early 2019.

About this project

This open dataset contains the block propagation times of approximately 15000 blocks on the Bitcoin network between November 2018 and March 2019. The data was collected from 9 locations (AWS EC2 instances), each connected to approximately 700 nodes at a time. Each of these locations ran Python custom code, to be cleaned up and open sourced soon.

This site is very much a work in progress. I will upload the data, along with some interactive visualisations and source code for everything in the coming months.

This is part of my masters thesis at the University of Melbourne, supervised by Prof. Peter Taylor.

If you're interested in this project, or would like any of the materials before I have a chance to polish off this website, please don't hesitate to email me at btc-page@aapelivuorinen.com.
